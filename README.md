# Kblog

**The easy way to learn web frameworks**

尝试使用多种语言和框架开发功能一致的博客系统，统一使用数据库PostgreSql，也是第一次尝试使用此数据库

#### Supported frameworks

**Python**
  * [Django](https://github.com/kaysen820/Kblog/tree/master/django)
  * Flask
  * Webpy

**PHP**
  * Laravel
  * ThinkPHP
  * Yii
  * CodeIgniter
  * CakePHP
  * Zend Framework

**Ruby**
  * Ruby on Rails

**Javascript (node.js)**
  * Express
