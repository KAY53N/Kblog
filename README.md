# Kblog

**The easy way to learn web frameworks**

尝试多种开发语言和框架写博客系统

统一使用数据库PostgreSQL

第一次用PHP之外语言写Web，也是第一次尝试使用此对象关系型数据库

#### Supported frameworks

**Python**
  * [Django](https://github.com/kaysen820/Kblog/tree/master/django)
  * Flask
  * Webpy

**PHP**
  * Laravel
  * ThinkPHP
  * Yii
  * CodeIgniter
  * CakePHP
  * Zend Framework

**Ruby**
  * Ruby on Rails

**Javascript (node.js)**
  * Express
